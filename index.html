Aventura das Cores

Arraste as flores para os vasos da mesma cor. Idade: 6–8 anos.

Palco de jogo

Fase 1
Acertos: 0
Arraste as flores para os vasos correspondentes.
Como jogar
  1. Arraste as flores para os vasos da mesma cor.
  2. Se acertar, a flor ficará no vaso.
  3. Complete todas para passar de fase.
Opções de acessibilidade
  • Contraste alto e botões grandes.
  • Modo lento para mais tempo de reação.
Protótipo: HTML5 + JavaScript — coordenação motora.
